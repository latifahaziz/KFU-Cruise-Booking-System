# 🚢 KFU Cruise Booking System

A desktop-based cruise management and booking system developed using **Java Swing** and **MySQL**, built as part of an Object-Oriented Programming course.

---

## 🚀 Overview

The KFU Cruise Booking System is a full desktop application that enables users to browse cruise destinations, select travel packages, and manage bookings through an interactive graphical interface.

The system demonstrates core concepts of **object-oriented programming, GUI development, and database integration** using Java.

---

## 💡 Key Features

### 👤 User Authentication

* User registration with validation (email & password constraints)
* Secure login system for registered users

### 🗺️ Cruise Booking System

* Browse multiple destinations (Jeddah, Bahrain, India, UK, USA, UAE)
* Select from different cabin packages:

  * **Standard** — basic accommodation
  * **Deluxe** — sea view with upgraded services
  * **Luxury** — premium suite with full features
* Automatic price calculation including **15% VAT**
* Payment options: Cash / Card

### 🛠️ Admin Features

* View all booking records
* Search bookings by last name
* Navigate records (Next / Previous)
* Delete booking entries

---

## 🧠 Technologies Used

* **Language:** Java
* **GUI:** Java Swing / AWT (NetBeans GUI Builder)
* **Database:** MySQL (JDBC)
* **IDE:** NetBeans
* **Build Tool:** Ant

---

## 🏗️ System Architecture

The system follows a layered structure:

* **Presentation Layer:** Java Swing UI (forms & event handling)
* **Logic Layer:** Booking logic, validation, and calculations
* **Data Layer:** MySQL database integration via JDBC
